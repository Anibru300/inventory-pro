# Inventory Pro Landing Page

Landing page moderna para Inventory Pro construida con Next.js 14 y Tailwind CSS.

## Características

- ⚡ Next.js 14 con App Router
- 🎨 Tailwind CSS para estilos
- ✨ Framer Motion para animaciones
- 📱 Diseño responsive
- 🔍 SEO optimizado
- 🚀 Export estático para hosting

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

El resultado estará en la carpeta `dist/`.
