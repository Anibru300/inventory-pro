# Inventory Pro Web

Frontend Vue.js 3 para Inventory Pro - Sistema de gestión de inventarios.

## Características

- Vue 3 + Composition API
- Pinia para estado global
- Vue Query para manejo de datos del servidor
- Tailwind CSS para estilos
- Gráficos interactivos
- Diseño responsive dark mode

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build para producción

```bash
npm run build
```

## Estructura

```
src/
├── assets/         # Estilos CSS globales
├── components/     # Componentes reutilizables
├── layouts/        # Layouts de la aplicación
├── router/         # Configuración de rutas
├── stores/         # Stores Pinia
├── views/          # Vistas de páginas
└── main.js         # Punto de entrada
```
